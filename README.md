# relative2alias

auto change relative path to alias path

eg: auto change '../../store' to '@/store'

## Installation

- Click on the Extensions icon (usually on the left-hand side of your editor).
- Search for "relative2alias".
- Find the extension in the list and click the install button.

## Usage

### Using the command palette:

- Bring up the command palette, and select "relative2alias".
- Select one of the commands mentioned below.
- Press [Enter] to confirm, or [Escape] to cancel.
